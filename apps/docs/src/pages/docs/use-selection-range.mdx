# useSelectionRange

**useSelectionRange** tracks the input selection range.

## Install

To start using **useSelectionRange**, you can install the `@react-awesome/hooks` library or you can import it directly from `@react-awesome/components` if you have installed it before. In your project directory, run
the following command to install the dependencies:

```sh npm2yarn
npm i @react-awesome/hooks
```

## Usage

import { useState } from 'react'
import { Container } from '../../components/Container'
import { useSelectionRange } from '@react-awesome/components'

export const Example = () => {
  const [ref, setRef] = useState(null)

const { caret } = useSelectionRange(ref)

return (

<>
  <input
    className="w-full border rounded-md px-3 py-2"
    ref={setRef}
    placeholder="Type something..."
  />
  <ul className="mt-2">
    <li>

        {caret.start.toString()}
      </code>
    </li>
    <li>
      <span className="font-medium">ðŸ‘‰ End</span>

        {caret.end.toString()}
      </code>
    </li>
  </ul>
</>
); };

<Container>
  <Example />
</Container>
